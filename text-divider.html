Float dividers with text

1. HTML:
<div class="divider2">
	<div class="divider2__text">I am a divider</div>
</div>
3. LESS:

/* --- MIXINS --- */

.inlineblock {
	display: inline-block;
	vertical-align: top;
	
	zoom: 1 !ie;
	display: inline !ie;
}

/* --- /MIXINS --- */

.divider2 {
	overflow: hidden; /* 1. Use overflow for limit divider, because lines will be very long */
}
.divider2__text {
	text-align: center;
	position: relative;
	padding: 10px;

	&:after, &:before; {
		content: "";
		.inlineblock;
		position: absolute;
		top: 10px;
		width: 2000px; /* Must be veeeeery looooong */
		height: 1px;
		background: #000;
	}
	
	/* Often useful method: */
	&:after { 
		left:100%;
	}

	&:before {
		right:100%;
	}
}
4. RESULT:
I am a divider
